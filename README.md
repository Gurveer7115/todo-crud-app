# Todo Application with CRUD Operations

## Aim
To build a full-stack todo application with basic CRUD operations.

## Course Outcomes
- CO1: Understand frontend and backend concepts
- CO2: Develop interactive UI using React
- CO3: Implement APIs using Node.js & Express
- CO4: Debug and test application
- CO5: Deploy full-stack application

## Features
- Add Todo
- View Todo
- Update Todo
- Delete Todo

## Tech Stack
- React.js
- Node.js
- Express.js

## Note
This project uses in-memory storage instead of MongoDB.
